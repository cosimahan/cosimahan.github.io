把博客从 WordPress 转移到了 Hexo

2015-08-02

      虽然WordPress是也很优秀的，不过我想换一换口味，便选择了 Hexo，它简单纯粹，静态网页还可以托管在 github page 上。
      迁移也是方便，用插件几条命令就搞定了，找了个自认为不错的主题，匆匆忙忙就完工了。
      还有顺手把图片啥的都放在的七牛上，感觉不错。
